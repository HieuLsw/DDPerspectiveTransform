## Installation

### CocoaPods

To install `DDPerspectiveTransform` via [CocoaPods](http://cocoapods.org), add the following line to your Podfile:

```
pod 'DDPerspectiveTransform'
```

## Requirements

- iOS 8.2
- Xcode 10, Swift 4.2
